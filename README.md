# UCloudTeach-Monolithic

#### 介绍
优云教在线教育平台单体架构版本，使用SpringBoot+Mybatis+Redis+Sa-Token搭建，此为第一版完整应用，基本功能都已实现，可用于毕设、练手Demo等。此版架构优点是构建难度较小，缺点是服务边界不明确、维护成本较高。
前端项目地址请关注Readme文件。
